# نظام التعرف على الوجه - Face Recognition System

## نظرة عامة

تم تطبيق نظام متقدم للتعرف على الوجه لحماية لوحة تحكم الأدمن. النظام يستخدم الكاميرا للتحقق من هوية الأدمن قبل السماح بالدخول.

## المميزات

✅ **التعرف على الوجه بالذكاء الاصطناعي**: استخدام مكتبة face-api.js المتقدمة
✅ **تسجيل تلقائي للمحاولات الفاشلة**: أي شخص غير مصرح له يتم تصويره وحفظ صورته
✅ **واجهة عربية كاملة**: تصميم dark mode احترافي
✅ **أمان عالي**: جميع البيانات محفوظة في Firebase
✅ **سهولة الاستخدام**: تسجيل الوجه مرة واحدة فقط

## التثبيت

### 1. تثبيت المكتبات

```bash
npm install
```

### 2. تحميل نماذج التعرف على الوجه

```bash
npm run download-models
```

أو يدوياً من: https://github.com/vladmandic/face-api/tree/master/model

### 3. رفع التطبيق

```bash
npm run build
```

## الاستخدام

### للأدمن - أول مرة (التسجيل)

1. اذهب إلى `/admin/login`
2. اضغط على "تسجيل وجه الأدمن"
3. أدخل اسم الأدمن
4. ضع وجهك أمام الكاميرا
5. انتظر العد التنازلي (3، 2، 1)
6. سيتم التقاط الصورة وتسجيل وجهك

### للأدمن - الدخول

1. اذهب إلى `/admin/login`
2. اضغط على "التحقق من الوجه"
3. ضع وجهك أمام الكاميرا
4. سيتم التحقق تلقائياً والدخول

### عرض محاولات الدخول الفاشلة

1. من لوحة التحكم، اذهب إلى "محاولات الدخول"
2. ستجد جميع الصور والمعلومات عن المحاولات الفاشلة

## قاعدة البيانات

### Collections في Firebase

1. **adminFaces**: وجوه الأدمن المسجلة
2. **unauthorizedAccessAttempts**: محاولات الدخول الفاشلة

## الملفات المهمة

- `src/services/faceRecognitionService.js`: خدمة التعرف على الوجه
- `src/components/FaceRecognitionLogin.js`: صفحة تسجيل الدخول
- `src/components/UnauthorizedAttempts.js`: صفحة عرض المحاولات الفاشلة
- `public/models/`: نماذج التعرف على الوجه

## ملاحظات مهمة

⚠️ **الكاميرا**: يجب السماح للمتصفح باستخدام الكاميرا
⚠️ **HTTPS**: النظام يعمل فقط على HTTPS (أو localhost)
⚠️ **الإضاءة**: تأكد من وجود إضاءة جيدة عند التسجيل/الدخول
⚠️ **المسافة**: ضع وجهك على مسافة مناسبة من الكاميرا
