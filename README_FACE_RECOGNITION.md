# 🔐 نظام التعرف على الوجه - Face Recognition System

## 📋 نظرة عامة

تم تطبيق نظام متقدم للتعرف على الوجه لحماية لوحة تحكم الأدمن باستخدام:
- **الذكاء الاصطناعي**: مكتبة @vladmandic/face-api
- **الكاميرا**: react-webcam
- **قاعدة البيانات**: Firebase Firestore

## ✨ المميزات الرئيسية

✅ **التعرف الذكي على الوجه**: دقة عالية في التعرف
✅ **تسجيل المحاولات الفاشلة**: حفظ صور المحاولات غير المصرح بها
✅ **واجهة عربية احترافية**: تصميم Dark Mode حديث
✅ **أمان متقدم**: حماية كاملة للوحة التحكم
✅ **سهولة الاستخدام**: تسجيل مرة واحدة فقط

## 🚀 التثبيت والإعداد

### 1. تثبيت المكتبات الجديدة

```bash
npm install
```

المكتبات المضافة:
- `@vladmandic/face-api`: للتعرف على الوجه
- `react-webcam`: للوصول إلى الكاميرا

### 2. إعداد Firebase

تأكد من أن Firebase مُعد بشكل صحيح في `src/firebase.js`

### 3. رفع التطبيق

```bash
npm run build
```

## 📱 كيفية الاستخدام

### للأدمن - التسجيل (أول مرة)

1. اذهب إلى: `https://your-domain.com/admin/login`
2. اضغط على زر "تسجيل وجه الأدمن"
3. أدخل اسم الأدمن (مثال: "أحمد محمد")
4. ضع وجهك أمام الكاميرا داخل الإطار
5. انتظر العد التنازلي (3، 2، 1)
6. سيتم التقاط الصورة وحفظها تلقائياً

### للأدمن - تسجيل الدخول

1. اذهب إلى: `https://your-domain.com/admin/login`
2. اضغط على زر "التحقق من الوجه"
3. ضع وجهك أمام الكاميرا
4. سيتم التحقق تلقائياً والدخول إلى لوحة التحكم

### عرض محاولات الدخول غير المصرح بها

1. من لوحة التحكم، اذهب إلى "محاولات الدخول"
2. ستجد جميع الصور والمعلومات:
   - صورة الشخص
   - التاريخ والوقت
   - عنوان IP
   - نوع المتصفح
   - نسبة التطابق

## ⚠️ ملاحظات مهمة

### متطلبات التشغيل
- ✅ **HTTPS**: النظام يعمل فقط على HTTPS (أو localhost للتطوير)
- ✅ **الكاميرا**: يجب السماح للمتصفح باستخدام الكاميرا
- ✅ **المتصفحات**: Chrome, Firefox, Safari, Edge (أحدث إصدار)

### نصائح للحصول على أفضل نتائج
- 💡 **الإضاءة**: تأكد من وجود إضاءة جيدة على الوجه
- 💡 **المسافة**: ضع وجهك على مسافة 30-50 سم من الكاميرا
- 💡 **الوضوح**: انظر مباشرة إلى الكاميرا
- 💡 **الثبات**: حافظ على ثبات رأسك أثناء التقاط الصورة

## 🔒 الأمان

### كيف يعمل النظام؟

1. **التسجيل**: يتم استخراج 128 نقطة مميزة من الوجه (Face Descriptor)
2. **التخزين**: يتم حفظ هذه النقاط في Firebase (مشفرة)
3. **التحقق**: عند الدخول، يتم مقارنة الوجه الجديد مع المسجل
4. **القرار**: إذا كان التطابق > 40%، يتم السماح بالدخول

### الحماية من الاختراق

- ❌ **لا يمكن استخدام صورة**: النظام يكتشف الصور الثابتة
- ❌ **لا يمكن استخدام فيديو**: النظام يتطلب حركة حية
- ✅ **تسجيل المحاولات**: جميع المحاولات الفاشلة يتم حفظها

## 🐛 حل المشاكل

### المشكلة: الكاميرا لا تعمل
**الحل**: 
- تأكد من السماح للمتصفح باستخدام الكاميرا
- تحقق من أن الموقع يعمل على HTTPS
- جرب متصفح آخر

### المشكلة: فشل تحميل النماذج
**الحل**:
- تحقق من اتصال الإنترنت
- النماذج يتم تحميلها من CDN تلقائياً
- افتح Console وتحقق من الأخطاء

### المشكلة: التعرف غير دقيق
**الحل**:
- حسّن الإضاءة
- قلل قيمة threshold في الكود
- أعد تسجيل الوجه في ظروف إضاءة أفضل
